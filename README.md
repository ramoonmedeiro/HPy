# HPy

HPy is a script created for educational purposes. It plots graphs of the radial and angular part of the wave function of the hydrogen atom, a topic covered in early quantum chemistry courses. 
Enjoy!!

# Installation

```
$ pip install horpy 
or
$ pip3 install horpy
```
